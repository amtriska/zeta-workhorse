The test files include:

- barbell.csv: A comma-separated (CSV) file containing an adjacency matrix for a barbell graph
- karate.gml: A GraphML (GML) file containing Wayne Zachary's "Zachary's Karate Club" graph
- test.py: A test script that executes the tropical_trace function on the karate.gml file

Below is is the original description of the Zachary's Karate Club GML file, downloaded from Mark Newman's network data web site (https://websites.umich.edu/~mejn/netdata/):

The file karate.gml contains the network of friendships between the 34 members of a karate club at a US university, as described by Wayne Zachary in 1977.  If you use these data in your work, please cite W. W. Zachary, An information flow model for conflict and fission in small groups, Journal of Anthropological Research 33, 452-473 (1977).
